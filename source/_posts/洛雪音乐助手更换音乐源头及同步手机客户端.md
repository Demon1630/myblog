# 一、更换音乐源

目前许多音乐客户端都需要开会员，经常听一首歌只能听一小段，而且不同歌曲在各音乐平台版权不一样，有时候还要各个平台切换就很麻烦，这里推荐一个音乐软件：[洛雪音乐助手](https://github.com/lyswhut/lx-music-desktop#readme)，软件开源，可以直接进链接到GetHub下载，或者到网盘下载：[https://strivefysfxyh.lanzoue.com/b01s3mv9a](https://strivefysfxyh.lanzoue.com/b01s3mv9a)密码:4f8h

软件界面很简洁但需要的功能都有，而且没有广告

![](https://cdn.jsdelivr.net/gh/Demon1630/picBed/img/20240305212649.png)

前期软件自带音乐源，可以下载各个平台高品质音乐，后面收到律师函没法再维护音乐源了，所以目前很多音乐也开始没法听和下载了，不过软件预留了音乐源接口，可以通过更换音乐源的方式继续使用，这里推荐使用六音的音乐源：[https://www.sixyin.com/8498.html](https://www.sixyin.com/8498.html)  可以到网盘下载：[https://strivefysfxyh.lanzoue.com/b01s3mv9a](https://strivefysfxyh.lanzoue.com/b01s3mv9a)密码:4f8h

下载后得到js文件格式的音乐源，然后到洛雪音乐助手上进行导入

![](https://cdn.jsdelivr.net/gh/Demon1630/picBed/img/20240305213755.png) ![](https://cdn.jsdelivr.net/gh/Demon1630/picBed/img/20240305213908.png)

将下载的js音乐源导入，然后选择该音乐源即可，就可以继续正常听和下载音乐了

# 二、开启同步功能

洛雪音乐助手除了有桌面版，还有手机版，可以到Github下载，或者到网盘下载：[https://strivefysfxyh.lanzoue.com/b01s3mv9a](https://strivefysfxyh.lanzoue.com/b01s3mv9a)密码:4f8h

软件带有同步功能，可以实现在电脑版与手机版间进行同步收藏的音乐，官方介绍：[https://lxmusic.toside.cn/desktop/faq/sync](https://lxmusic.toside.cn/desktop/faq/sync)

这里介绍使用独立服务器进行同步，采用docker部署，[https://hub.docker.com/r/wbsu2003/lx-music-sync-server](https://hub.docker.com/r/wbsu2003/lx-music-sync-server)

使用如下命令把镜像拉取到服务器上

```
docker pull wbsu2003/lx-music-sync-server
```

参照如下命令进行配置

```
# 新建文件夹 lx-music-sync-server 和 子目录
mkdir -p /volume2/docker/lx-music-sync-server/{data,logs}

# 进入 lx-music-sync-server 目录
cd /volume2/docker/lx-music-sync-server

# 运行容器
docker run -d \
   --restart unless-stopped \
   --name lx-music-sync-server \
   -p 9527:9527 \
   -v $(pwd)/data:/server/data \
   -v $(pwd)/logs://server/logs \
   -e LX_USER_user1=mypassword123 \
   wbsu2003/lx-music-sync-server
```

记得在防火墙中放行容器端口，默认9527

然后到软件中进行设置：

![](https://cdn.jsdelivr.net/gh/Demon1630/picBed/img/20240305220031.png) 查看状态为已连接则表明已经完成配置，同理，手机端也进行类似配置，就可以实现在手机端和电脑端的收藏列表进行同步了
