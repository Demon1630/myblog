使用宝塔面板管理树莓派还是很方便的，这里介绍一下如何安装宝塔面板，树莓派安装好原装系统后，使用一键安装宝塔面板即可：

```
wget -O install.sh http://download.bt.cn/install/install-ubuntu.sh && sudo bash install.sh
```

即可进入安装流程，或者到宝塔面板官网查看自己系统对应的安装脚本[https://www.bt.cn/bbs/thread-19376-1-1.html](https://www.bt.cn/bbs/thread-19376-1-1.html)

等待安装完成即可，宝塔面板打开时需要使用的是树莓派的ip地址，不是安装完成后显示的ip，但端口号还是那个

![](http://www.demon1630.cn/wp-content/uploads/2020/01/1578220830-树莓派13.png)

 

如果安装过程中经常出现报错说缺这缺那的话，可以尝试使用宝塔独立安装

```
wget -O install.sh http://download.bt.cn/install/install_panel.sh &&sudo bash install.sh
```

 

更新：

目前使用树莓派官方的固件，一直出现安装不了宝塔的问题，这里推荐使用树莓派爱好者基地的系统文件，该系统优化比较好，可以正常安装：

[https://github.com/openfans-community-offical/Debian-Pi-Aarch64/blob/master/README\_zh.md](https://github.com/openfans-community-offical/Debian-Pi-Aarch64/blob/master/README_zh.md)

切换root账号后再安装宝塔

wget -O install.sh http://download.bt.cn/install/install-ubuntu\_6.0.sh && bash install.sh

 

安装后要是忘记登录地址和密码，可以在ssh里输入

bt default
