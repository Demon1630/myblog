newifi 3 不仅可以刷高恪固件，还可以刷华硕老毛子固件，目前 newifi 3 可以刷的固中，华硕老毛子固件是最为稳定的，刷入比例也是最高的。

newifi 3 已经在刷入不死 Breed 的状态下，要刷入各种系统，是非常简单的，要刷路由器地址就得先进入进入 Breed Web 恢复控制台，先按住 RESET 按键，再插入电源。

**newifi3 华硕老毛子最新固件下载地址：[https://opt.cn2qq.com/padavan/](https://link.zhihu.com/?target=https%3A//opt.cn2qq.com/padavan/)。**在刷路由器之前，有网络先把固件下载好。

打开浏览器输入 192.168.1.1，进入 Breed Web 恢复控制台刷系统， 目前只能通过有线连接的方式进行，因为进入 BreedWeb 恢复控制台后，无线功能是没有打开的。

![](https://pic4.zhimg.com/v2-f477579b9bb07543fc49427f2d59869b_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-551cca3ceae371464adf2f0d1f278a61.jpg)

进入 Breed Web 恢复控制台后，先不要急着进行升级，先在控制台点击恢复出厂设置。

路由如果从一个系统刷入另一个系统，最好先恢复出厂设置，这样也能保持在刷入系统之前，是最初始状态。

![](https://pic4.zhimg.com/v2-21770f561ca249e9029eda4e1564a3eb_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-8d779d1cde313adcd08a634ebf2b3407.jpg)

恢复出厂设置后，不需要进行任何操作，会自动跳到 BreedWeb 恢复控制台。然后在固件前面打勾，再点击”浏览”，跳出固件选择窗口。

如果没有下载到最新的 newifi 3 固件，也不要紧，那就选择电脑上之前的 newifi 3 版本，最新的版本可以在联网后进行下载，在本地进行升级。

![](https://pic4.zhimg.com/v2-92f508b89440caa2497175c39c48c813_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-ec65c691ffc8f97a085b1f01f86d615c.jpg)

如果没有下载到最新的 newifi 3 固件，也不要紧，那就选择电脑上之前的 newifi 3 版本，最新的版本可以在联网后进行下载，在本地进行升级。

![](https://pic3.zhimg.com/v2-15e725b7ee6741e0d3db20c50837713a_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-fc7578e5d195b79557a0aa6f642b3bc3.jpg)

选择固件后，点击上传。

![](https://pic2.zhimg.com/v2-f18857e5122c6d6b2cb86a0d217e1b65_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-ebfcb3ee9dc380d2b6b7e2e535ec141e.jpg)

固件上传完成后，到 Breed Web 恢复控制台更新确认界面，点击更新。

![](https://pic1.zhimg.com/v2-b62d77d3834c70dff0284a333c94df7c_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-813d7f99fec3e0528e6375a2bfc00c84.jpg)

然后就到更新操作进度界面，静待更新完成。

![](https://pic2.zhimg.com/v2-0e6a4bdc32545507f0b08b0142b55579_b.jpg)

![](<data:;base64,\<svg xmlns='http://www.w3.org/2000/svg' width='900' height='293'\>\</svg\>>)

更新完成后，停止在这个界面，设备正在重启。

![](https://pic2.zhimg.com/v2-789663537228097cf42d1e5eef6e3d19_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-9f77626bb1cd5d47df4f4efea4abcbe2.jpg)

重启后，看电脑右下角网络联接符已经出现，在浏览器中输入 192.168.123.1，并回车。

提示要求输入用户名及密码，华硕老毛子系统默认的用户名 admin，密码 admin，点击确认。

![](https://pic1.zhimg.com/v2-f776b0b2c3defd16f0668c35b3e49c54_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-d1ac0ae2e2d26fe665b7e386db4d389f.jpg)

然后就进入路由器设置界面。

![](https://pic3.zhimg.com/v2-9b40250de49d342cf4621082e0109226_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-26c35ced16d9671c71ce2d2bfb21ef6c.jpg)

如果固件不是最新的，可以先下载最新的固件，然后在本地进行升级。

点击左侧菜单”系统管理”，然后再点击右侧”固件升级”，就到了本地固件升级界面，下载最新的固件，都可以在此进行升级。

![](https://pic3.zhimg.com/v2-f3a7d64141bd59b4d8167a611820359a_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-b9c3b68a27a37caeb4a4950e18fe2246.jpg)

点击选择文件，然后在弹出窗口中选择最新固件。

![](https://pic4.zhimg.com/v2-79980fa5258a4d3fc093505fbee3dfef_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-8e2f2be4fa4d73313582c370b39674cb.jpg)

![](https://pic1.zhimg.com/v2-93afe013eeea5500e00779602ce13550_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-25be3cb45590b69f9f5f6d2456443cc8.jpg)

选择 newifi 3 固件后，点击上传。

![](https://pic4.zhimg.com/v2-807e891f3514fa767657d115ad637a13_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-0cb119440829cc9a4e6236c650d8b97f.jpg)

系统会进入升级界面。

![](https://pic2.zhimg.com/v2-6956a030343f94549c8500ae9104fda5_b.jpg)

![](http://www.demon1630.cn/wp-content/uploads/2021/01/frc-0fcccf0eb66bfe71d397d53f1f906c69.jpg)

更新完成后，会自动到初始界面，然后就可以进行设置，如 SSH 功能等。

newifi 3 华硕老毛子固件主要特点就是稳定，长时间开机不会自动重启，网络延时较低等特点，但是因为“漏油”而翻过车，这将严重影响无线传输。2.4G 无线很差，所以一般将 newifi 3 作为主路由。如果你有动手能力，可将路由拆下，将散热硅上析出来的硅油清洁下，情况会有所改善。
