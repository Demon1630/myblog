# 搬瓦工+LNMP搭建网站

本文主要讲的是如何使用搬瓦工来搭建属于自己的网站，其实搭建自己的网站并没有想象中那么难，有人可能会问要自己的网站干嘛？当然是好处多多了，比如说可以建博客来记录自己的生活，可以掌握相关技术，等等。。。

   **下面就开始讲如何开始用搬瓦工搭建自己的网站吧！**

首先你需要购买搬瓦工的VPS，有了搬瓦工的VPS，可以说是好处多多，可以用来[搭梯子](https://baike.baidu.com/item/%E7%BF%BB%E5%A2%99/754773?fr=aladdin)，至于搭梯子干嘛？就自己发挥了，如何购买请看上一篇[**如何购买搬瓦工**](https://www.demon1630.cf/%e6%90%ac%e7%93%a6%e5%b7%a5vps%e8%b4%ad%e4%b9%b0/)，这里就不多说了。

在购买完搬瓦工的VPS之后，就可以开始搭建自己的网站了，为方便搭建，这里使用的是[LNMP一键安装包](https://lnmp.org/install.html)，可查看安装包详情。

首先使用ssh连接到搬瓦工后台，如何使用ssh，请查看文章**如何使用ssh连接搬瓦工**后台，进入后台之后，

首先创建screen会话：

```
# screen -S lamp
```

如果显示：

```
screen: command not found
```

可以使用命令：

```
# apt-get install screen
```

此目的主要是为防止安装过程中出现异常导致中断，可以输入

```
# screen -r lamp
```

  来恢复安装界面。

之后开始下载LNMP最新安装包：

```
#  wget -c http://soft.vpser.net/lnmp/lnmp1.6-full.tar.gz && tar -zxf lnmp1.6-full.tar.gz && cd lnmp1.6-full && ./install.sh lamp
```

此时安装开始，按照安装选项一步步进行就行，选项一般默认即可，设置的数据库密码需牢记，后面需要用到。如果数据库密码忘记了，可以进行重置，详情看重置MySQL root密码。

当出现绿字 "Press any key to install...or Press Ctrl+c to cancel" 后，按回车键确认开始安装。

安装时间可能较长，通常在一个小时内安装完成，安装完成会出现"Install lnmp V1.6 completed! enjoy it "
