### 阿里云上宝塔面板定时运行python脚本

```
要是能有一台不会关机的电脑能够帮我定时运行程序就好了
```

小时候就想要这么个东西 然后去年双十一时候买了阿里云的学生机，感觉挺神奇，装了面板，然后我就有了我的第一个web页面。 本文介绍如何在宝塔面板中定时运行python程序（其他语言类似）

1.登陆面板 2.找到面板左侧的【计划任务】 3.在跳转的页面添加任务 ![](http://www.demon1630.cn/wp-content/uploads/2021/05/frc-896ddef399b659f76250cffbd23913cf.png) ![](http://www.demon1630.cn/wp-content/uploads/2021/05/frc-ac5cdb8c6a3c7f4aac67cb43f9080322.png)

- 任务类型默认是Shell脚本，后面也有备注，还可以定时执行网站备份，数据库备份之类，这里默认就好
- 任务名称就是给自己本地看的名称，相当于一个备注
- 执行周期，每天、每小时、每几分钟之类
- 脚本内容 python /www/wwwroot/域名/文件绝对路径

这个命令就是调用python运行某个文件，可以在远程链接中自己运行这条指令，其他的语言同理。

- 确认各项数据无误
- 点击确认添加任务 ![](http://www.demon1630.cn/wp-content/uploads/2021/05/frc-ff8e3ab0708224bb169a422ee48a74b2.png) 这样就能够定时执行某个文件啦可以各种骚操作

然后我就设计了一个爬虫的脚本，对接上了钉钉的机器人，每天自动整理数据钉钉还是好用的说
